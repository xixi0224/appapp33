# 笔记重命名功能使用说明

## ✅ 已实现的功能

### 1. **长按修改笔记名称**
- 在"我的笔记"页面，长按任意笔记卡片
- 弹出修改名称对话框
- 输入新的笔记名称（最多50个字符）
- 点击"确定"保存

### 2. **实时更新**
- 修改成功后立即刷新笔记列表
- 所有页面显示的都是最新的笔记名称
- 不会再出现乱码问题

### 3. **输入验证**
- ✅ 不能为空
- ✅ 最多50个字符
- ✅ 自动去除首尾空格

---

## 📱 使用方法

### 步骤1: 进入笔记列表
```
首页 → 我的 → 我的笔记
或
底部导航 → 学习 → 我的笔记 → 查看全部
```

### 步骤2: 长按笔记
- 找到想要修改的笔记
- **长按**该笔记卡片（约1秒）
- 弹出"修改笔记名称"对话框

### 步骤3: 输入新名称
- 在输入框中输入新的笔记名称
- 例如：
  - ❌ 旧名称：`音频转文字 2024-01-15 14:32:45`
  - ✅ 新名称：`高等数学第三章笔记`
  - ✅ 新名称：`英语听力练习-Unit5`
  - ✅ 新名称：`Python学习心得`

### 步骤4: 保存
- 点击"确定"按钮
- 等待"修改成功"提示
- 笔记列表自动刷新，显示新名称

## 🔧 技术实现

### 前端（微信小程序）

**笔记卡片 - 添加长按事件**
```xml
<view class="note-card" 
      bindlongpress="showRenameModal"
      data-note-id="{{item.id}}"
      data-note-title="{{item.title}}">
```

**重命名弹窗**
```xml
<view class="magic-modal" wx:if="{{showRenameModal}}">
  <input 
    class="rename-input" 
    value="{{renameValue}}"
    bindinput="onRenameInput"
    placeholder="请输入新的笔记名称"
    maxlength="50"
  />
</view>
```

**JS逻辑**
```javascript
// 显示重命名弹窗
showRenameModal(e) {
  const noteId = e.currentTarget.dataset.noteId
  const noteTitle = e.currentTarget.dataset.noteTitle
  
  this.setData({
    showRenameModal: true,
    currentNoteId: noteId,
    renameValue: noteTitle
  })
}

// 确认重命名
confirmRename() {
  wx.request({
    url: `${app.globalData.baseUrl}/api/notes/${currentNoteId}/title`,
    method: 'PUT',
    data: { title: newTitle },
    success: (res) => {
      // 刷新笔记列表
      this.loadAllNotes()
    }
  })
}
```

### 后端（FastAPI）

**API接口**
```python
@router.put("/notes/{note_id}/title")
def update_note_title(note_id: int, title_data: dict = Body(...)):
    """更新笔记标题"""
    new_title = title_data.get("title", "").strip()
    
    # 验证
    if not new_title:
        raise HTTPException(status_code=400, detail="笔记名称不能为空")
    
    if len(new_title) > 50:
        raise HTTPException(status_code=400, detail="笔记名称不能超过50个字符")
    
    # 更新数据库
    cursor.execute(
        "UPDATE zhinote_notes SET title = %s, updated_at = NOW() WHERE id = %s",
        (new_title, note_id)
    )
    
    return {"code": 0, "message": "修改成功"}
```

## 🐛 常见问题

### Q1: 长按没有反应？
**原因**: 长按时间不够  
**解决**: 按住笔记卡片约1秒钟

### Q2: 修改后没有立即显示？
**原因**: 网络延迟  
**解决**: 等待1-2秒，列表会自动刷新

### Q3: 提示"笔记名称不能为空"？
**原因**: 输入框为空或只有空格  
**解决**: 输入有效的笔记名称

### Q4: 提示"笔记名称不能超过50个字符"？
**原因**: 名称太长  
**解决**: 缩短名称到50个字符以内

### Q5: 修改后其他页面还是旧名称？
**原因**: 其他页面缓存了旧数据  
**解决**: 重新进入该页面即可看到新名称

## 💡 使用建议

### 命名规范建议
1. **课程笔记**: `课程名 + 章节` 
   - 例如：`高等数学-第三章`、`英语-Unit5`

2. **音频转写**: `主题 + 日期`
   - 例如：`会议记录-0429`、`课堂讲座-0428`

3. **学习笔记**: `主题 + 类型`
   - 例如：`Python基础-笔记`、`算法总结`

4. **文件导入**: `文件名简化`
   - 例如：`线性代数讲义`、`复习资料`

### 避免的名称
- ❌ 纯数字：`123456`
- ❌ 特殊字符：`@#$%^&*`
- ❌ 过长描述：`今天老师讲了很多关于高等数学的内容...`（超过50字）
