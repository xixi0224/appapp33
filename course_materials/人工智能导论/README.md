## 文档格式说明

### Markdown文件 (.md)
- 完整的课程讲义，包含理论讲解、示例、思考题
- 适合导入知识图谱系统进行AI分析
- 包含8个章节，覆盖AI核心知识

### 文本文件 (.txt)
- `知识点总结.txt`：全课程核心知识点汇总
- `练习题.txt`：各章节选择题、填空题、简答题
- 适合快速浏览和搜索

### Word文件 (.docx)
- `课程大纲.docx`：完整的课程大纲
- 包含课程概述、教学内容、考核方式、进度安排等
- 使用python-docx生成

## 使用方法

### 导入知识图谱系统
1. 打开知识图谱页面
2. 点击"上传文件"按钮
3. 选择本目录下的任意文件或全部文件
4. 系统会自动分析文件内容并生成知识图谱

### 测试建议
- 可以单独上传某一章的讲义进行测试
- 也可以批量上传所有文件进行完整课程测试
- 上传后系统会自动提取知识点并生成图谱
