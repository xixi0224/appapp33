# 第1章 人工智能概述

## 1.1 人工智能的定义与发展历程

### 1.1.1 什么是人工智能

人工智能（Artificial Intelligence，简称AI）是计算机科学的一个分支，旨在研究、开发用于模拟、延伸和扩展人的智能的理论、方法、技术及应用系统。

人工智能的核心目标是使计算机系统能够：
- **感知**：获取和理解外部信息（视觉、听觉、语言等）
- **认知**：处理和推理信息（学习、记忆、决策等）
- **行动**：执行任务（运动、交流、控制等）

### 1.1.2 人工智能的发展阶段

**第一阶段：孕育期（1943-1955）**
- 1943年：McCulloch-Pitts神经元模型
- 1948年：香农信息论
- 1950年：图灵测试

**第二阶段：诞生期（1956-1974）**
- 1956年：达特茅斯会议，AI正式诞生
- 1957年：Rosenblatt感知机
- 1961年：通用问题求解器(GPS)
- 1969年：SRI机器人SHAKEY

**第三阶段：瓶颈期（1974-1980）**
- 计算能力不足
- 数据匮乏
- 算法局限性

**第四阶段：知识工程期（1980-1993）**
- 专家系统兴起
- 知识表示研究
- 机器学习方法发展

**第五阶段：连接主义复兴（1993-2012）**
- 神经网络重新兴起
- 支持向量机
- 随机森林

**第六阶段：深度学习时代（2012年至今）**
- 2012年：AlexNet赢得ImageNet比赛
- 大规模数据和计算能力提升
- 深度学习模型突破

## 1.2 人工智能的分类

### 1.2.1 按能力分类

**弱人工智能（Narrow AI）**
- 专注于特定任务的AI系统
- 如：语音助手、图像识别、推荐系统
- 当前主流的AI应用

**强人工智能（Strong AI）**
- 具备人类般综合智能的AI系统
- 能够理解、学习、推理各种任务
- 目前尚未实现

**超人工智能（Super AI）**
- 超越人类智慧的AI系统
- 仅存在于理论和科幻作品中

### 1.2.2 按技术方法分类

**符号主义（Symbolic AI）**
- 基于逻辑和规则的方法
- 知识表示与推理
- 专家系统

**连接主义（Connectionism）**
- 基于神经网络的方法
- 模拟人脑神经元连接
- 深度学习

**行为主义（Behaviorism）**
- 基于强化学习的方法
- 通过与环境交互学习
- 智能体理论

## 1.6 本章小结

本章介绍了人工智能的基本概念、发展历程、分类方法、应用领域、关键技术以及面临的挑战。人工智能已从理论研究走向广泛应用，深刻改变着各个行业。

## 关键知识点

1. **人工智能定义**：研究模拟、延伸和扩展人类智能的技术
2. **AI发展阶段**：孕育期、诞生期、瓶颈期、知识工程期、连接主义复兴、深度学习时代
3. **AI分类**：弱AI、强AI、超AI
4. **技术方法**：符号主义、连接主义、行为主义
5. **核心技术**：机器学习、深度学习、知识图谱、NLP
