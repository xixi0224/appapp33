# 第2章 机器学习基础

## 2.1 机器学习概述

### 2.1.1 什么是机器学习

机器学习（Machine Learning，简称ML）是人工智能的核心技术之一，它使计算机系统能够从数据中学习，自动改进性能，而无需显式编程。

机器学习的核心思想：
- 通过数据训练模型
- 从经验中学习模式
- 做出预测或决策

### 2.1.2 机器学习与传统编程的区别

| 传统编程 | 机器学习 |
|---------|---------|
| 人工编写规则 | 从数据中学习规则 |
| 输入：规则+数据 | 输入：数据 |
| 输出：答案 | 输出：模型 |
| 适用于规则明确的问题 | 适用于复杂模式问题 |

## 2.2 机器学习的类型

### 2.2.1 监督学习

**定义**：使用标记数据进行训练，模型学习输入与输出之间的映射关系。

**典型任务**：
- 分类：预测离散标签（如：猫/狗）
- 回归：预测连续数值（如：房价）

**常见算法**：
- 线性回归
- 逻辑回归
- 决策树
- 支持向量机
- 神经网络

### 2.2.2 无监督学习

**定义**：使用未标记数据进行训练，模型自动发现数据中的模式和结构。

**典型任务**：
- 聚类：将相似数据分组
- 降维：减少特征维度
- 异常检测：识别异常数据

**常见算法**：
- K-Means聚类
- 层次聚类
- PCA（主成分分析）
- 自编码器

### 2.2.3 半监督学习

**定义**：结合少量标记数据和大量未标记数据进行训练。

**应用场景**：
- 数据标注成本高
- 部分数据有标签

### 2.2.4 强化学习

**定义**：智能体通过与环境交互，获得奖励或惩罚来学习最优策略。

**核心要素**：
- 智能体（Agent）
- 环境（Environment）
- 状态（State）
- 动作（Action）
- 奖励（Reward）

**常见算法**：
- Q-Learning
- Deep Q-Network (DQN)
- Policy Gradient
- Actor-Critic

## 2.3 机器学习的基本流程

### 2.3.1 数据收集与预处理

**数据收集**：
- 数据源选择
- 数据量评估
- 数据质量检查

**数据预处理**：
- 数据清洗：处理缺失值、异常值
- 特征提取：从原始数据中提取有用特征
- 特征缩放：标准化、归一化
- 数据划分：训练集、验证集、测试集

### 2.3.2 模型选择与训练

**模型选择**：
- 根据任务类型选择模型
- 考虑模型复杂度和性能需求

**模型训练**：
- 定义损失函数
- 选择优化算法
- 调整超参数
- 防止过拟合

### 2.3.3 模型评估与优化

**评估指标**：
- 分类：准确率、精确率、召回率、F1分数
- 回归：MAE、MSE、RMSE、R²

**优化方法**：
- 超参数调优
- 特征工程
- 集成学习

### 2.3.4 模型部署与监控

**部署方式**：
- 在线服务
- 离线预测
- 嵌入式部署

**监控维护**：
- 性能监控
- 数据漂移检测
- 模型更新

## 2.4 特征工程

### 2.4.1 特征选择

**方法**：
- 过滤法：基于统计指标筛选特征
- 包裹法：通过模型性能评估特征
- 嵌入法：利用模型自身选择特征

### 2.4.2 特征变换

**方法**：
- 标准化（Standardization）
- 归一化（Normalization）
- 离散化
- 编码（One-hot编码、标签编码）

### 2.4.3 特征构造

**方法**：
- 特征组合
- 统计特征
- 领域知识特征

## 2.5 模型评估指标

### 2.5.1 分类评估指标

**混淆矩阵**：
- 真阳性（TP）
- 真阴性（TN）
- 假阳性（FP）
- 假阴性（FN）

**常用指标**：
- 准确率 = (TP+TN)/(TP+TN+FP+FN)
- 精确率 = TP/(TP+FP)
- 召回率 = TP/(TP+FN)
- F1分数 = 2×精确率×召回率/(精确率+召回率)

### 2.5.2 回归评估指标

**常用指标**：
- MAE（平均绝对误差）
- MSE（均方误差）
- RMSE（均方根误差）
- R²（决定系数）

## 2.6 过拟合与欠拟合

### 2.6.1 过拟合

**定义**：模型在训练集上表现很好，但在测试集上表现很差。

**原因**：
- 模型过于复杂
- 训练数据过少
- 噪声数据影响

**解决方法**：
- 增加训练数据
- 简化模型
- 正则化（L1、L2）
- 交叉验证

### 2.6.2 欠拟合

**定义**：模型在训练集和测试集上表现都很差。

**原因**：
- 模型过于简单
- 特征不足
- 训练不足

**解决方法**：
- 增加模型复杂度
- 增加特征
- 延长训练时间

本章介绍了机器学习的基本概念、分类、流程、特征工程、评估指标以及过拟合问题。机器学习是人工智能的核心技术，掌握这些基础知识对于理解和应用AI至关重要。

3. **基本流程**：数据预处理→模型选择→训练→评估→部署
4. **特征工程**：特征选择、变换、构造
5. **评估指标**：准确率、精确率、召回率、F1、MAE、MSE、R²
6. **过拟合**：训练集好、测试集差；欠拟合：都差
