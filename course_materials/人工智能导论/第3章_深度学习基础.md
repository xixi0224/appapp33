# 第3章 深度学习基础

## 3.1 深度学习概述

### 3.1.1 什么是深度学习

深度学习（Deep Learning）是机器学习的一个分支，使用多层神经网络来模拟人脑的学习过程，能够自动从原始数据中学习特征表示。

**深度学习的特点**：
- 多层神经网络结构
- 自动特征学习
- 需要大量数据和计算资源
- 端到端学习

### 3.1.2 深度学习与传统机器学习的区别

| 传统机器学习 | 深度学习 |
|-------------|---------|
| 需要人工特征工程 | 自动学习特征 |
| 模型较简单 | 模型复杂（多层） |
| 数据量需求较小 | 需要大量数据 |
| 计算资源需求低 | 需要GPU加速 |

## 3.2 神经网络基础

### 3.2.1 神经元模型

**人工神经元**：
- 接收多个输入
- 计算加权和
- 通过激活函数输出

**数学表达式**：
y = f(Σw_i * x_i + b)

其中：
- x_i：输入
- w_i：权重
- b：偏置
- f：激活函数
- y：输出

### 3.2.2 激活函数

**作用**：引入非线性，使神经网络能够学习复杂模式。

**常用激活函数**：

**Sigmoid**：
- 输出范围：(0, 1)
- 问题：梯度消失
- 公式：σ(x) = 1 / (1 + e^(-x))

**ReLU（Rectified Linear Unit）**：
- 输出范围：[0, ∞)
- 优点：计算简单，缓解梯度消失
- 问题：神经元死亡
- 公式：ReLU(x) = max(0, x)

**Tanh**：
- 输出范围：(-1, 1)
- 公式：tanh(x) = (e^x - e^(-x)) / (e^x + e^(-x))

**Leaky ReLU**：
- 解决ReLU神经元死亡问题
- 公式：LeakyReLU(x) = max(αx, x)

## 3.3 前馈神经网络

### 3.3.1 网络结构

**组成**：
- 输入层：接收原始数据
- 隐藏层：提取特征（可以有多层）
- 输出层：产生最终预测

**全连接网络**：每一层的神经元与下一层所有神经元相连。

### 3.3.2 前向传播

**定义**：数据从输入层经过隐藏层，最终到达输出层的过程。

**计算过程**：
1. 输入层接收数据
2. 每一层计算加权和
3. 通过激活函数
4. 传递到下一层
5. 输出层产生预测

### 3.3.3 反向传播

**定义**：计算损失函数对各参数的梯度，用于更新权重。

**算法步骤**：
1. 前向传播计算预测值
2. 计算损失函数
3. 从输出层反向计算梯度
4. 使用链式法则传播梯度
5. 更新权重和偏置

## 3.4 卷积神经网络

### 3.4.1 CNN概述

**卷积神经网络（Convolutional Neural Network，CNN）**：专门用于处理网格数据（如图像）的神经网络。

**核心思想**：
- 局部感受野
- 权值共享
- 池化层降维

### 3.4.2 卷积层

**卷积操作**：使用滤波器（卷积核）在输入上滑动，计算局部加权和。

**参数**：
- 滤波器数量
- 滤波器大小
- 步长
- 填充

**优点**：
- 减少参数数量
- 保留空间结构
- 平移不变性

### 3.4.3 池化层

**作用**：降低特征图尺寸，减少计算量，增强鲁棒性。

**类型**：
- 最大池化（Max Pooling）：取区域最大值
- 平均池化（Average Pooling）：取区域平均值

### 3.4.4 典型CNN架构

**LeNet-5**（1998年）：
- 输入层 → 卷积层 → 池化层 → 卷积层 → 池化层 → 全连接层 → 输出层

**AlexNet**（2012年）：
- 更深的网络结构
- 使用ReLU激活函数
- 使用Dropout防止过拟合
- 使用GPU加速

**VGGNet**（2014年）：
- 更深的网络（16/19层）
- 使用3x3小卷积核
- 统一的网络结构

**ResNet**（2015年）：
- 残差连接解决梯度消失
- 超深网络（152层）

## 3.5 循环神经网络

### 3.5.1 RNN概述

**循环神经网络（Recurrent Neural Network，RNN）**：专门用于处理序列数据的神经网络。

**特点**：
- 处理可变长度序列
- 具有记忆能力
- 时间步之间共享参数

### 3.5.2 RNN结构

**隐藏状态**：
h_t = f(W * x_t + U * h_{t-1} + b)

**输出**：
y_t = g(V * h_t + c)

### 3.5.3 RNN的问题

**梯度消失/爆炸**：
- 长序列训练困难
- 无法捕捉长期依赖

**解决方案**：
- LSTM
- GRU

### 3.5.4 LSTM

**长短期记忆网络（Long Short-Term Memory，LSTM）**：改进的RNN，能够学习长期依赖。

**门控机制**：
- 遗忘门：决定丢弃哪些信息
- 输入门：决定存储哪些新信息
- 输出门：决定输出哪些信息

### 3.5.5 GRU

**门控循环单元（Gated Recurrent Unit，GRU）**：简化版LSTM。

**特点**：
- 合并遗忘门和输入门
- 参数更少
- 训练更快

## 3.6 Transformer架构

### 3.6.1 Transformer概述

**Transformer**：基于自注意力机制的序列模型，不依赖RNN结构。

**核心创新**：
- 自注意力机制
- 多头注意力
- 位置编码
- 残差连接和层归一化

### 3.6.2 自注意力机制

**计算过程**：
1. 计算查询（Query）、键（Key）、值（Value）
2. 计算注意力权重
3. 加权求和得到输出

**优点**：
- 并行计算
- 直接建模任意位置依赖
- 可解释性强

### 3.6.3 Transformer结构

**编码器**：
- 多头注意力层
- 前馈神经网络
- 残差连接和层归一化

**解码器**：
- 掩码多头注意力层
- 编码器-解码器注意力层
- 前馈神经网络
