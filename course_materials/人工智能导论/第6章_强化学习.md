# 第6章 强化学习

## 6.1 强化学习概述

### 6.1.1 什么是强化学习

强化学习（Reinforcement Learning，简称RL）是机器学习的一个分支，智能体通过与环境交互，获得奖励或惩罚来学习最优策略。

**RL的特点**：
- 试错学习
- 延迟奖励
- 序列决策
- 与环境交互

### 6.1.2 强化学习与监督学习的区别

| 监督学习 | 强化学习 |
|---------|---------|
| 有标记数据 | 无标记数据 |
| 即时反馈 | 延迟反馈 |
| 独立样本 | 序列决策 |
| 分类/回归 | 策略优化 |

## 6.2 强化学习基本概念

### 6.2.1 智能体与环境

**智能体（Agent）**：
- 学习和决策的主体
- 接收状态，执行动作

**环境（Environment）**：
- 智能体所处的外部世界
- 接收动作，返回状态和奖励

### 6.2.2 马尔可夫决策过程

**马尔可夫决策过程（MDP）**：
- 状态集合S
- 动作集合A
- 状态转移概率P(s'|s,a)
- 奖励函数R(s,a,s')
- 折扣因子γ

**马尔可夫性质**：
未来状态仅依赖当前状态和动作，与历史无关。

### 6.2.3 策略与价值函数

**策略（Policy）**：
- 从状态到动作的映射
- π(a|s)：在状态s下选择动作a的概率

**价值函数（Value Function）**：
- 状态价值V(s)：从状态s开始的期望累积奖励
- 动作价值Q(s,a)：在状态s执行动作a后的期望累积奖励

## 6.3 动态规划

### 6.3.1 策略评估

**定义**：给定策略，计算价值函数。

**迭代方法**：
V(s) = Σπ(a|s)ΣP(s'|s,a)[R(s,a,s') + γV(s')]

### 6.3.2 策略改进

**定义**：基于当前价值函数改进策略。

**贪心策略**：
π'(s) = argmax_a ΣP(s'|s,a)[R(s,a,s') + γV(s')]

### 6.3.3 策略迭代

**算法**：
1. 初始化策略
2. 策略评估：计算价值函数
3. 策略改进：更新策略
4. 重复直到收敛

### 6.3.4 值迭代

**算法**：
1. 初始化价值函数
2. 更新价值函数：V(s) = max_a ΣP(s'|s,a)[R(s,a,s') + γV(s')]
3. 重复直到收敛
4. 提取策略

## 6.4 Monte Carlo方法

### 6.4.1 蒙特卡洛策略评估

**定义**：通过采样完整轨迹来估计价值函数。

**优点**：
- 不需要环境模型
- 无偏估计

**缺点**：
- 方差大
- 需要完整轨迹

### 6.4.2 蒙特卡洛控制

**算法**：
1. 初始化Q函数和策略
2. 生成轨迹
3. 更新Q函数
4. 改进策略
5. 重复

## 6.5 时序差分学习

### 6.5.1 TD(0)算法

**定义**：使用时序差分误差更新价值函数。

**TD目标**：
G_t = R_{t+1} + γV(S_{t+1})

**TD误差**：
δ_t = R_{t+1} + γV(S_{t+1}) - V(S_t)

**更新规则**：
V(S_t) ← V(S_t) + αδ_t

### 6.5.2 SARSA算法

**定义**：基于动作价值函数的时序差分算法。

**更新规则**：
Q(S_t, A_t) ← Q(S_t, A_t) + α(R_{t+1} + γQ(S_{t+1}, A_{t+1}) - Q(S_t, A_t))

**特点**：
- 在线策略算法
- 使用实际动作更新

### 6.5.3 Q-Learning算法

**定义**：基于动作价值函数的离线策略算法。

**更新规则**：
Q(S_t, A_t) ← Q(S_t, A_t) + α(R_{t+1} + γmax_a Q(S_{t+1}, a) - Q(S_t, A_t))

**特点**：
- 离线策略算法
- 使用最优动作更新
- 更稳定收敛

## 6.6 函数近似

### 6.6.1 线性函数近似

**定义**：使用线性函数近似价值函数。

**优点**：
- 计算简单
- 收敛保证

**缺点**：
- 表达能力有限

### 6.6.2 非线性函数近似

**定义**：使用神经网络近似价值函数。

**优点**：
- 表达能力强
- 处理复杂问题

**缺点**：
- 训练不稳定
- 需要探索-利用平衡

## 6.7 深度强化学习

### 6.7.1 Deep Q-Network (DQN)

**核心创新**：
- 经验回放
- 目标网络
- 卷积神经网络

**算法流程**：
1. 初始化Q网络和目标网络
2. 收集经验存储到回放缓冲区
3. 随机采样经验批量
4. 计算目标Q值
5. 更新Q网络
6. 定期更新目标网络

### 6.7.2 Policy Gradient

**定义**：直接优化策略函数。

**目标函数**：
J(θ) = E[Σγ^t R_t]

**梯度估计**：
∇J(θ) = E[Σγ^t ∇logπ_θ(a_t|s_t) * G_t]

**优点**：
- 连续动作空间
- 策略搜索直接

### 6.7.3 Actor-Critic

**定义**：结合价值函数和策略函数。

**架构**：
- Actor：策略网络
- Critic：价值网络

**更新**：
- Actor：使用优势函数更新
- Critic：使用TD误差更新

**优点**：
- 减少方差
- 提高稳定性

## 6.8 探索与利用

### 6.8.1 ε-greedy策略

**定义**：
- 以概率ε随机选择动作（探索）
- 以概率1-ε选择最优动作（利用）

**衰减策略**：
ε随时间逐渐减小

### 6.8.2 Upper Confidence Bound (UCB)

**定义**：平衡探索和利用的算法。

**选择策略**：
a = argmax_a [Q(a) + c * sqrt(log(t)/N(a))]

### 6.8.3 Thompson Sampling

**定义**：基于贝叶斯方法的探索策略。

**采样策略**：
- 为每个动作维护奖励分布
- 从分布中采样
- 选择采样值最大的动作

## 6.9 本章小结

本章介绍了强化学习的基本概念、MDP、动态规划、Monte Carlo方法、时序差分学习、函数近似和深度强化学习。强化学习是AI领域最具挑战性的方向之一，在游戏、机器人、自动驾驶等领域有重要应用。

3. **策略**：从状态到动作的映射
