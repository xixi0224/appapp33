# 人工智能导论

- **课程代码**: CS301
- **学分**: 3
- **学期**: 秋季

## 课程简介

本课程介绍人工智能的基本概念、方法和应用，涵盖机器学习、深度学习、自然语言处理、计算机视觉等核心领域。

## 章节大纲

### 第一章 人工智能概述

- 1.1 人工智能的定义与发展历程
- 1.2 人工智能的研究领域
- 1.3 人工智能的应用

### 第二章 机器学习基础

- 2.1 机器学习概述
- 2.2 监督学习算法
- 2.3 模型评估与选择

### 第三章 深度学习

- 3.1 神经网络基础
- 3.2 卷积神经网络
- 3.3 循环神经网络

### 第四章 自然语言处理

- 4.1 文本表示方法
- 4.2 Transformer架构
- 4.3 预训练语言模型

### 第五章 计算机视觉

- 5.1 图像处理基础
- 5.2 图像分类与目标检测
- 5.3 图像分割与生成

### 第六章 强化学习

- 6.1 强化学习基础
- 6.2 强化学习算法
- 6.3 强化学习应用

### 第七章 知识图谱

- 7.1 知识图谱概述
- 7.2 知识图谱应用

### 第八章 人工智能伦理与安全

- 8.1 AI伦理问题
- 8.2 AI安全问题
- 8.3 AI发展趋势
