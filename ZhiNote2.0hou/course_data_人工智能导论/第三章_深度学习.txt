########################################
3.1 神经网络基础
########################################

人工神经网络（ANN）是模仿生物神经网络结构和功能的计算模型。

神经元模型：
- 输入：x1, x2, ..., xn
- 权重：w1, w2, ..., wn
- 偏置：b
- 激活函数：f(z)，z = sum(wi*xi) + b

常用激活函数：
1. Sigmoid：f(z) = 1 / (1 + exp(-z))
   - 将输出映射到[0,1]
   - 梯度消失问题

2. ReLU：f(z) = max(0, z)
   - 计算简单，缓解梯度消失
   - 可能导致神经元死亡

3. Tanh：f(z) = (exp(z) - exp(-z)) / (exp(z) + exp(-z))
   - 将输出映射到[-1,1]
   - 比sigmoid收敛更快

神经网络结构：
- 输入层：接收原始数据
- 隐藏层：进行特征变换和学习
- 输出层：产生最终预测

前向传播：数据从输入层流向输出层
反向传播：根据误差更新权重

########################################
3.2 卷积神经网络
########################################

卷积神经网络（CNN）是专门设计用于处理网格数据（如图像）的深度学习模型。

卷积操作：
- 使用滤波器（kernel）对输入进行滑动窗口计算
- 提取局部特征，如边缘、纹理
- 参数共享：同一个滤波器在整个输入上共享

CNN基本组件：
1. 卷积层（Convolutional Layer）
   - 提取特征图
   - 多个滤波器提取不同特征

2. 池化层（Pooling Layer）
   - 降低特征图尺寸
   - 减少参数数量，防止过拟合
   - 最大池化、平均池化

3. 全连接层（Fully Connected Layer）
   - 将特征图展开为向量
   - 进行分类或回归

经典CNN架构：
- LeNet：早期CNN，手写数字识别
- AlexNet：深度CNN突破，ImageNet冠军
- VGG：统一的3x3卷积核设计
- ResNet：残差连接，解决深度网络训练问题
- GoogLeNet：Inception模块，多尺度特征融合

########################################
3.3 循环神经网络
########################################

循环神经网络（RNN）是处理序列数据的深度学习模型。

RNN特点：
- 具有记忆能力，可以利用历史信息
- 递归结构，适合处理时间序列、文本等顺序数据

RNN结构：
- 隐藏状态：保存历史信息
- 当前输入：当前时刻的数据
- 当前输出：基于隐藏状态和输入的预测

RNN公式：
h_t = f(W_hh * h_{t-1} + W_xh * x_t + b_h)
y_t = g(W_hy * h_t + b_y)

RNN变体：
1. LSTM（长短期记忆网络）
   - 门控机制：输入门、遗忘门、输出门
   - 记忆细胞：保存长期信息
   - 解决梯度消失问题

2. GRU（门控循环单元）
   - 简化的LSTM
   - 更新门和重置门

3. Bi-LSTM（双向LSTM）
   - 同时利用前向和后向信息
   - 适合需要上下文的任务
