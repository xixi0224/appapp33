## 方案一：使用网络穿透（最快）

### 适用场景
- ✅ 比赛现场演示
- ✅ 快速测试
- ✅ 不想迁移数据库

### 步骤

#### 1. 使用 cpolar（国内推荐）

**下载安装**
```
官网：https://www.cpolar.com
选择：Windows版本
```

**配置步骤**
```bash
# 1. 注册cpolar账号
# 2. 登录客户端
# 3. 创建隧道

配置：
- 隧道名称：ZhiNote-Backend
- 协议：HTTP
- 本地地址：localhost:8000
- 端口：随机或自定义
```

**启动服务**
```bash
# 终端1：启动后端
cd "d:\tingting\xixi\计算机设计大赛\ZhiNote2.0\ZhiNote2.0hou"
uvicorn app.main:app --reload --port 8000

# 终端2：启动cpolar
cpolar http 8000

# 获得公网地址
# 例如：https://abc123.cpolar.cn
```

#### 2. 配置微信小程序

**修改前端配置**
```javascript
// ZhiNote2.0qian/app.js
App({
  globalData: {
    baseUrl: 'https://abc123.cpolar.cn', // 替换为你的cpolar地址
    userInfoUpdated: false
  }
})
```

**注意事项**
- ⚠️ cpolar免费版地址每次重启会变
- ⚠️ 需要每次手动启动
- ⚠️ 电脑必须保持开机

## 方案二：部署到云服务器

### 适用场景
- ✅ 正式比赛
- ✅ 长期使用
- ✅ 需要稳定服务

### 步骤

#### 1. 购买云服务器

**推荐平台**
```
阿里云：https://www.aliyun.com
- 学生机：9.5元/月
- 凭edu邮箱或学生证认证

腾讯云：https://cloud.tencent.com
- 学生机：类似价格
- 有免费试用
```

**配置建议**
```
CPU：1核
内存：1GB
系统：Ubuntu 20.04 / CentOS 7
带宽：1Mbps（够用）
```

#### 2. 服务器环境配置

**SSH连接服务器**
```bash
ssh root@你的服务器IP
# 输入密码
```

**安装环境**
```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装Python 3.8+
sudo apt install python3 python3-pip python3-venv -y

# 安装MySQL
sudo apt install mysql-server -y
sudo mysql_secure_installation

# 安装Nginx（反向代理）
sudo apt install nginx -y

# 安装Supervisor（进程管理）
sudo apt install supervisor -y
```

#### 3. 部署后端代码

**上传代码**
```bash
# 方法1：使用Git
git clone https://github.com/your-username/ZhiNote2.0.git

# 方法2：使用SCP
scp -r "d:\tingting\xixi\计算机设计大赛\ZhiNote2.0\ZhiNote2.0hou" root@服务器IP:/opt/

# 方法3：使用FTP工具（FileZilla）
```

**配置后端**
```bash
cd /opt/ZhiNote2.0hou

# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 配置数据库
mysql -u root -p
```

```sql
-- 创建数据库
CREATE DATABASE zhinote CHARACTER SET utf8mb4;

-- 创建用户
CREATE USER 'zhinote'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON zhinote.* TO 'zhinote'@'localhost';
FLUSH PRIVILEGES;

-- 导入数据
USE zhinote;
source /opt/ZhiNote2.0hou/sql/create_users.sql;
source /opt/ZhiNote2.0hou/sql/knowledge_graph.sql;
-- ... 导入其他SQL文件
```

**修改数据库配置**
```python
# app/db.py
DB_CONFIG = {
    "host": "localhost",
    "port": 3306,
    "user": "zhinote",
    "password": "your_password",
    "database": "zhinote",
    "charset": "utf8mb4"
}
```

#### 4. 配置进程管理

**创建Supervisor配置**
```bash
sudo nano /etc/supervisor/conf.d/zhinote.conf
```

```ini
[program:zhinote]
command=/opt/ZhiNote2.0hou/venv/bin/uvicorn app.main:app --host 0.0.0.0 --port 8000
directory=/opt/ZhiNote2.0hou
user=root
autostart=true
autorestart=true
stderr_logfile=/var/log/zhinote.err.log
stdout_logfile=/var/log/zhinote.out.log
```

**启动服务**
```bash
sudo supervisorctl reread
sudo supervisorctl update
sudo supervisorctl start zhinote

# 查看状态
sudo supervisorctl status
```

#### 5. 配置Nginx反向代理

```bash
sudo nano /etc/nginx/sites-available/zhinote
```

```nginx
server {
    listen 80;
    server_name your_domain.com; # 或服务器IP

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

```bash
# 启用配置
sudo ln -s /etc/nginx/sites-available/zhinote /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
```

#### 6. 配置域名和HTTPS（可选）

```bash
# 安装certbot
sudo apt install certbot python3-certbot-nginx -y

# 获取SSL证书
sudo certbot --nginx -d your_domain.com
```

## 方案三：部署到免费云平台

### 使用 Render.com

#### 1. 准备代码

**创建 render.yaml**
```yaml
services:
  - type: web
    name: zhinote-backend
    env: python
    buildCommand: pip install -r requirements.txt
    startCommand: uvicorn app.main:app --host 0.0.0.0 --port $PORT
    envVars:
      - key: DATABASE_URL
        sync: false
      - key: DASHSCOPE_API_KEY
        sync: false
```

**修改数据库配置**
```python
# app/db.py
import os

def get_conn():
    import pymysql
    
    # Render环境变量
    database_url = os.getenv("DATABASE_URL")
    
    if database_url:
        # 解析URL
        # mysql://user:password@host:port/dbname
        ...
    else:
        # 本地开发
        return pymysql.connect(**DB_CONFIG)
```

#### 2. 部署

```
1. 代码推送到GitHub
2. 登录 https://render.com
3. New Web Service
4. 连接GitHub仓库
5. 配置环境变量
6. 点击Deploy
```

#### 3. 配置数据库

**使用TiDB Cloud**
```
1. 注册 https://tidbcloud.com
2. 创建集群
3. 获取连接信息
4. 导入SQL文件
5. 在Render配置DATABASE_URL
```

## 常见问题

### Q1: 微信小程序能用http吗？
**A**: 不能，必须使用https。开发阶段可以在详情→本地设置中勾选"不校验合法域名"。

### Q2: 数据库太大怎么办？
**A**: 
```bash
# 导出数据库
mysqldump -u root -p zhinote > zhinote_backup.sql

# 压缩
gzip zhinote_backup.sql

# 上传到服务器后导入
mysql -u root -p zhinote < zhinote_backup.sql
```

### Q3: 比赛时网络不稳定怎么办？
**A**: 
1. 准备离线演示视频
2. 准备本地运行环境
3. 准备截图PPT

### Q4: 小程序审核被拒怎么办？
**A**: 
1. 查看拒绝原因
2. 修改后重新提交
3. 申请加急审核（说明是比赛项目）
